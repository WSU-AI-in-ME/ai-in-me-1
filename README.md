# Wayne State University - AI in Mechanical Engineering I: Fundamentals and Machine Learning

**Instructors:** Eunseob Kim (Fall 2026) & Longsheng Feng (Winter 2027)

## About
This course is the first of a two-semester sequence designed to systematically introduce AI to mechanical engineering students. Its primary objective is to bridge traditional mechanical engineering with foundational data science. Through a hands-on approach to Python programming, data preprocessing, and core machine learning algorithms, students will develop the essential computational skills required for advanced AI applications.
